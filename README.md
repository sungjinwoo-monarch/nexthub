# Overview

NextHub is a modern GitHub developer discovery platform designed for developers, recruiters, students, and open-source contributors.

Search any GitHub username and instantly explore beautiful developer analytics, repositories, contribution activity, organizations, languages, and much more—all wrapped inside a premium Bento Grid interface.

---

# Features

- GitHub Profile Search
- Beautiful Developer Dashboard
- Premium Bento Grid UI
- Repository Explorer
- Contribution Heatmap
- Programming Language Statistics

# Installation

```bash
git clone https://github.com/YOUR_USERNAME/nexthub.git
```

```bash
cd nexthub
```

```bash
npm install
```

```bash
npm run dev
```

---

# Build

```bash
npm run build
```

---

# Environment Variables

Create a `.env` file

```env
VITE_GITHUB_TOKEN=your_github_token
```

(Optional if you want higher GitHub API rate limits.)
